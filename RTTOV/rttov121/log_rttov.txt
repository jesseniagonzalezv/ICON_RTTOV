=========================================================================
 RTTOV compilation script
=========================================================================

This script compiles RTTOV, optionally with HDF5.
It should be run from the src/ directory.

Much RTTOV functionality is available without HDF5: only those
features listed below require HDF5 to compile.

If compiling with HDF5 (recommended) you must first edit the file
build/Makefile.local with the location of the library.

RTTOV features which require the HDF5 library:
- Reading HDF5 coefficient files
- Emissivity/BRDF atlases
- RTTOV GUI

In addition f2py must be installed for the Python interface and the RTTOV GUI.
